  7. def dispatch(qName: String, pmmlnode: PmmlNode): Unit

    Select the appropriate PmmlExecNode generator (a PmmlExecNodeGenerator) for the supplied xml element name by locating its PmmlExecNodeGenerator (the 'qName' is the element name key), and dispatch it.

    Select the appropriate PmmlExecNode generator (a PmmlExecNodeGenerator) for the supplied xml element name by locating its PmmlExecNodeGenerator (the 'qName' is the element name key), and dispatch it. The returned node is added to the syntax tree being constructed. The dispatcher is passed as an argument so that the generators are free to access the 'raw' and 'cooked' syntax trees being built via the Context method.
