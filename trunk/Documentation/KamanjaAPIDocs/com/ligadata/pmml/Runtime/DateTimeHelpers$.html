DateTimeHelpers

Related Doc: package runtime

object DateTimeHelpers extends LogTrait

DateTimeHelpers contains several builtin date & time parse functions. Pre-built patterns of the common formats (hopefully) for timestamps, dates, and times are used to determine the number of millisecs since the epoch (all date and time types are represented in this fashion). Those values then can be converted to other useful formats using the core udf library in the PmmlUdfs object.
