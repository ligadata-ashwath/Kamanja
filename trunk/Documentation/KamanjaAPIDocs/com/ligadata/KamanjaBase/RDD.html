class RDD[T] extends AnyRef

RDD - One of the basic classes for Kamanja development. It is basically a collection of individual scala RDDObjects. If you are working with Java, JavaRDD will support the same nethod.

Note that calling toJavaRDD on any RDD[Object] will return a JavaRDD[T]
